# GestureAI – AI Powered Gesture and Voice Assistant

## Overview

GestureAI is a computer vision and voice-controlled desktop assistant developed using Python, OpenCV, MediaPipe, and Speech Recognition.

The system recognizes hand gestures through a webcam and performs desktop automation tasks such as launching applications, controlling volume, opening websites, and executing voice commands.

## Features

* Real-time hand tracking using MediaPipe
* Gesture recognition using finger landmark detection
* Voice command activation through gestures
* Desktop automation
* Browser control
* Volume control
* Application launching
* Human-computer interaction system

## Technology Stack

* Python
* OpenCV
* MediaPipe
* SpeechRecognition
* PyAutoGUI
* PyTTSx3

## Supported Gestures

| Gesture       | Action                   |
| ------------- | ------------------------ |
| Palm          | Activate Voice Assistant |
| Victory       | Open VS Code             |
| One Finger    | Volume Up                |
| Three Fingers | Volume Down              |
| Fist          | Exit Program             |
